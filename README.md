# Shahyan Abbas — Personal Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, and journey as a Software Engineering student.

## ✨ Features

- 🌗 Dark/Light mode toggle
- 📱 Fully responsive design
- 🎨 Glassmorphism & gradient aesthetics
- ⚡ Smooth scroll animations
- ⌨️ Typing effect hero section
- 📊 Animated skill progress bars
- 📬 Contact form UI
- 🔍 SEO optimized

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. That's it — no build tools required!

## 📁 Structure

```
shahyan/
```
